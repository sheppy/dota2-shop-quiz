# Dota 2 Shop Quiz

A JavaScript Dota 2 Shop Quiz!

## Setup

Install *npm* modules:
```bash
cd path/to/dota2-shop-quiz/
npm install
```


## Fetch data

Download the latest item data (as these are not stored in version control)

```bash
npm run fetch
```


## Build the code

```bash
npm run build
```


## Start server

Run the server

```bash
npm start
```
